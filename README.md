## Selected Projects

### SentinelAI — Medical NLP and MedDRA Coding Assistance

A completed AI Engineering capstone exploring the assignment of free-text medical-device safety reports to standardized MedDRA Preferred Terms.

The system combines:

- lexical and vector retrieval in PostgreSQL
- reciprocal-rank fusion
- cross-encoder reranking
- a local Qwen LLM through Ollama
- a FastAPI service and reviewer workflow
- MLflow-based experiment tracking
- Grafana monitoring
- pytest and GitHub Actions
- human-in-the-loop review and auditable result storage

The project was developed as a portfolio system for regulated-data scenarios. The original Hetzner deployment is deliberately offline, while the source code, architecture, evaluation results, and presentation remain available.
